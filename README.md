coach AI
Description

coach AI is a comprehensive web application and Telegram bot platform for collecting, managing, and recommending flirting lines. The system features both Python Flask and Node.js Express implementations, providing users with the ability to browse, search, and receive AI-powered recommendations for conversation starters. The platform includes data import/export utilities, duplicate removal tools, and statistical analysis features.

A Python/Node.js project that delivers an intelligent flirting line coach with web interface, Telegram bot integration, and AI-powered recommendations.

Features

AI-powered flirting line recommendations based on context and user preferences

Web application with EJS templating for browsing and managing lines

Telegram bot integration for on-the-go access to flirting advice

Database management with MySQL for storing and retrieving lines

CSV import/export functionality for bulk data operations

Web scraping utilities for collecting new flirting lines

Duplicate detection and removal tools

Statistical analysis of line effectiveness and usage

Random line generator for spontaneous suggestions

Multi-language support (Python Flask and Node.js implementations)

Technologies Used

Node.js 16 or higher

Flask web framework

Express.js web framework
